## 💈 Landing Page - Barbearia

Este projeto consiste em uma landing page moderna para uma barbearia, desenvolvida com foco em apresentação visual, experiência do usuário e conversão de clientes.

A página foi projetada para destacar os principais serviços oferecidos, como cortes de cabelo e cuidados com a barba, além de transmitir profissionalismo, estilo e confiança. O layout valoriza a estética masculina com um design limpo, elegante e bem estruturado.

## 🚀 Funcionalidades

- 📌 Seção de apresentação com destaque visual
- ✂️ Exibição dos serviços oferecidos
- 🧔 Foco em estilo, cuidado e atendimento personalizado
- ⭐ Área de depoimentos para gerar credibilidade
- 📞 Seção de contato para facilitar o agendamento

## 🎨 Design e Experiência

O design foi pensado para ser responsivo e adaptável a diferentes dispositivos (desktop, tablet e celular), garantindo uma navegação fluida e intuitiva.

A estrutura segue o padrão de páginas modernas, conduzindo o usuário de forma estratégica até a ação principal (contato ou agendamento), algo essencial para negócios locais como barbearias .

## 🛠️ Tecnologias Utilizadas

HTML5 🟧
CSS3 🟦
JavaScript 🟨

## 🎯 Objetivo

O objetivo do projeto é simular uma presença digital profissional para uma barbearia, ajudando a atrair novos clientes, apresentar serviços e fortalecer a marca no ambiente online.
